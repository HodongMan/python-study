# 일급 함수

  

## 일급함수의 조건

```
    변수나 데이터 구조 안에 할당할 수 있다.
    파라미터로 전할 수 있다.
    반환값(return value)로 전할 수 있다.
    런타임에 생성할 수 있다.
```

## High Order Function

```
    하나 이상의 함수를 인수로 취한다.
    함수를 결과로 반환한다.
```
